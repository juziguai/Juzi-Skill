# 更新日志

本文件记录 Juzi Skill 市场的用户可见变化。正式发布使用 market-vYYYY.MM.DD.N annotated tag。

## [Unreleased]

### 新增

- 统一项目注册表、发布预检、隔离 Canary、发布凭证、健康报告和回滚计划。
- Windows/Linux CI、CodeQL、周度市场健康检查及显式市场发布工作流。
- 公共仓库安全、贡献、兼容、弃用、资产和长期运维治理文档。
- MIT License 及其在统一项目注册表中的许可证声明。

### 变更

- 市场目录、README 目录和验证范围改由项目注册表统一管理。
- 发布通道拆分为 main/beta 候选与 stable 正式通道。
- 子项目测试按 Git 变更范围选择，同时保留全局市场和安全门禁。

### 修复

- 防止新增插件目录、纯 Skill 或测试时遗漏市场验证器的硬编码清单。
- 防止插件包已变更但版本/cachebuster 未更新后进入发布流程。
- 防止公开前遗漏完整 Git 历史秘密扫描、资产预算和回滚入口。
- 修复 Windows 非 UTF-8 系统区域下发布预检无法读取子进程输出或写入中文诊断的问题。
- 修复公开仓库治理错误 PATCH 自动可用的 Advanced Security，导致首次公开切换回滚的问题。
- 修复 Ruleset 写入后列表短暂不可见时被误判失败的问题，并增加有限轮询失败注入。
